# Описание проблем

## Область
Приложение для создания мемов на базе React/Vite. Обнаруженные проблемы относятся к логике работы канваса, устойчивости к ошибкам, доступности и UX.

## Проблемы
1. **Дублирование текста при скачивании**
   - Симптом: при скачивании изображение содержит текст с удвоенной обводкой/жирностью.
   - Причина: текст уже нарисован на канвасе из эффекта отрисовки, а `downloadImage()` рисует его еще раз перед экспортом.
   - Влияние: ухудшение качества итогового изображения.

2. **Краш при битых данных в localStorage**
   - Симптом: приложение падает на старте, если `localStorage.meme` содержит некорректный JSON.
   - Причина: `JSON.parse` вызывается без try/catch.
   - Влияние: невозможность открыть приложение при неконсистентных данных.

3. **Перетаскивание текста не работает на тач-устройствах**
   - Симптом: на мобильных устройствах текст нельзя перемещать.
   - Причина: используются только мышиные события (`mousemove`, `mouseup`, `onMouseDown`).
   - Влияние: критичное ограничение UX на телефонах/планшетах.

4. **Некорректные id для color picker**
   - Симптом: оба color picker имеют одинаковые `id/name`.
   - Причина: компонент `ColorPicker` использует фиксированные `id/name`.
   - Влияние: нарушение доступности и возможная путаница фокуса/лейблов.

5. **Изменения параметров текста не попадают в историю**
   - Симптом: `Undo` не откатывает смену цвета/размера/шрифта/обводки.
   - Причина: `pushToHistory()` вызывается не во всех изменениях состояния.
   - Влияние: непоследовательное поведение истории и редактирования.
