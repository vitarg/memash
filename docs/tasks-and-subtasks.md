# Задачи и подзадачи

## 1. Исправить дублирование текста при скачивании
- 1.1 Проанализировать текущий поток отрисовки канваса и экспорт.
- 1.2 Сохранить базовое состояние канваса до отрисовки текста.
- 1.3 Рисовать текст для экспорта единым проходом.
- 1.4 Восстановить состояние канваса после экспорта.
- 1.5 Проверить результат на разных наборах текста.

## 2. Устойчивость к битым данным localStorage
- 2.1 Обернуть `JSON.parse` в `try/catch`.
- 2.2 При ошибке удалить `meme` и продолжить с пустым состоянием.
- 2.3 Проверить сценарий с некорректным JSON.

## 3. Поддержка тач-перетаскивания текста
- 3.1 Перевести логику drag на Pointer Events.
- 3.2 Добавить `pointermove`, `pointerup`, `pointercancel` обработчики.
- 3.3 Реализовать `setPointerCapture`/`releasePointerCapture`.
- 3.4 Проверить в мобильном эмуляторе.

## 4. Уникальные id для color picker
- 4.1 Добавить проп `id` или использовать `useId()` внутри компонента.
- 4.2 Прокинуть уникальные значения для каждого инстанса.
- 4.3 Проверить корректность `label`.

## 5. История для изменений параметров текста
- 5.1 Вставить `pushToHistory()` перед изменением параметров выбранного текста.
- 5.2 Рассмотреть ограничение истории при частых изменениях слайдеров (по необходимости).
- 5.3 Проверить `Undo`/`Redo` для цвета, размера, шрифта, обводки.
